## How to Use

1.  **Open the Plugin**: Click the "Full Focus" icon in the Tintero sidebar.
2.  **Setup Your Session**:
    *   **Select a File**: Choose an existing file to continue writing or type a new name to create one.
    *   **Set Duration**: Pick a preset time or enter a custom duration.
    *   **Toggle Background**: Turn the constellation animation on/off if you prefer absolute black.
3.  **Start Writing**: Click "Begin Session". After a 3-second countdown, write until the timer stops!
4.  **Review**: When time is up, view your session statistics and see your text seamlessly integrated into your project.
